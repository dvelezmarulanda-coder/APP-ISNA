# ISNA Boutique - Wholesale & Retail Platform 💎

¡Bienvenido a la plataforma oficial de **ISNA Boutique**! Una solución de comercio electrónico moderna, elegante y funcional diseñada para ofrecer una experiencia de lujo tanto a clientes al detal como a mayoristas.

![ISNA Boutique Preview](https://via.placeholder.com/1200x600/001f3f/b8943f?text=ISNA+BOUTIQUE+PREMIUM+STORE)

## ✨ Características Principales

### 🏪 Experiencia de Compra Dual
- **Modo Detal**: Interfaz limpia y elegante para el cliente final.
- **Modo Mayorista**: Acceso exclusivo mediante código secreto (`ISNAMAYORISTA`) que desbloquea precios especiales y visualización de descuentos en tiempo real.

### 🛒 Carrito de Compras Avanzado
- Sistema de carrito lateral (Drawer) con gestión de cantidades.
- **Checkout Inteligente**: Formulario de captura de datos de envío (Nombre, Teléfono, Dirección, Ciudad) integrado antes del pago.
- Totalización dinámica basada en el modo de compra activo.

### 🛡️ Panel Administrativo Robusto
- **Dashboard**: Estadísticas diarias de ingresos, ventas y stock crítico.
- **Inventario**: Control total de productos, categorías y existencias (Vitrina vs Bodega).
- **Gestión de Ventas**: Seguimiento detallado de pedidos con estados de despacho (Pendiente / Despachado).
- **Contacto Directo**: Botón de integración con WhatsApp para contactar al cliente al instante desde el panel.

### 📱 Diseño Premium & Responsivo
- Estética **Navy & Gold** inspirada en marcas de lujo.
- Totalmente optimizado para dispositivos móviles.
- Animaciones fluidas y micro-interacciones de alta calidad.

## 🚀 Tecnologías Utilizadas
- **React 18** + **Vite** (Frontend ultra rápido)
- **Tailwind CSS** (Diseño moderno y responsivo)
- **Lucide Icons** (Iconografía minimalista)
- **Zustand Pattern** (Gestión de estado global optimizada)
- **LocalStorage** (Persistencia de datos sin necesidad de servidor externo)

## 🛠️ Instalación y Uso

1. Clona este repositorio:
   ```bash
   git clone https://github.com/dvelezmarulanda-coder/ISNA-WEB.git
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
